About the Project

The Unauthorized Access Monitoring System (U.A.M.S) is a Final Year Project developed by students from GMI, aiming to enhance data security by detecting and preventing unauthorized USB access.

How It Works

01 Device Detection

Windows agent captures all USB device insertions instantly.

02 AI Analysis

AI engine processes Defender results & applies rules.

03 Real-time Action

Access is allowed/blocked & SMS alerts sent to admin.

Frequently Asked Questions

How is it different?

U.A.M.S uses symbolic AI to analyze Defender data intelligently.

Does it work offline?

Yes, with cached policies. AI and SMS features need internet.

Supported Windows versions?

Windows 10 and 11 (with Defender).

Can rules be customized?

Yes, fully configurable thresholds and exceptions.
