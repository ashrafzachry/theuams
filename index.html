Why U.A.M.S?

AI-Powered Decisions

Symbolic AI analyzes Windows Defender scan results to make instant allow/deny decisions without human intervention.

SMS Alerts

Receive instant SMS notifications when unauthorized USB devices are detected or blocked.

Smart Whitelisting

Granular control with custom rules, device fingerprinting, and automatic policy enforcement.

Centralized Dashboard

Monitor all USB activity, view threat logs, and manage policies from a single interface.

How It Works

01 Device Detection

Windows agent detects USB device insertion and captures device information.

02 AI Analysis

Symbolic AI processes Windows Defender scan results and applies security policies.

03 Real-time Action

Device is instantly allowed or blocked with SMS notification sent to administrators.

Frequently Asked Questions

How does U.A.M.S differ from traditional USB blockers?

U.A.M.S uses symbolic AI to make intelligent decisions based on Windows Defender scan results.

Does it work offline?

Yes, the Windows agent caches policies and can operate offline, but AI and SMS features require internet.

What Windows versions are supported?

Windows 10 and 11 with Windows Defender enabled.

Can I customize the AI decision rules?

Yes, the dashboard allows configuration of custom rules, thresholds, and exceptions.

Is this ready for production use?

No. This is a Final Year Project demonstration for educational purposes only.
